1. Last used value in python console
```bash
>>> 10 + 10
20
>> _ * 2
40
```

2. Make number more readable
```python
number: int = 1_000_000
```

3. Unimportant values
```python
for _ in range(3):
    print('hello')
```

4. Unimportant exception
```python
try:
  result = 1/0
except ZeroDivisionError as _:
   print('Handled.')
```

5. Unpacking
```python
sample_tuple = (1,2,3,4)
a, _, b, _ = sample_tuple
print(a,b) # 1, 3
```

6. Super Unpacking
```python
sample_tuple = (1, 2, 3, 4, 5)
a, *_, b = sample_tuple
print(a, b)  #1, 5
```

7. protected
```python
from uuid import uuid4
class User:
    def __init__(self):
        self._id = uuid4()
    def _get_id(self):
        return self._id

user = User()
# typing user. will not show _get_id from IDE
user._get_id()
```

8. Private
```python
def __init__(self):
    self.__id = uuid4()
def __get_id(self):
    return self.__id
```

9. Reserved names
```python
if_ = 10 
class_ = "class"
str_ = "str"
```

10. dundor method
```python
class Fruit:
    def __init__(self):
        self.id = uuid4()
    def __str__(self):
        return f"Fruit: {self.id}"
fruit = Fruit()
print(fruit)
>>> Fruit: 6075f5a9-24bf-4df3-b1ba-3fea2100dd4d
```
